# Aurtsy Caregiver Support System

A comprehensive system for caregivers to capture information and analyze behavior using AI.

## Project Structure

- **`ios-app/`**: Native iOS application (SwiftUI) for data capture and sharing.
- **`backend/`**: FastAPI (Python) server for API, data management, and orchestration.
- **`ai-worker/`**: Python service for ML inference (Computer Vision, Behavior Analysis).
- **`legacy_backup/`**: Archived React/Node.js prototype.
